Vue

Contient le résultat d'une requète SQL

Exemple

CREATE VIEW mavue AS

SELECT * FROM recette;

(stock le résultat de la requète dans la vue mavue)

.

SELECT * FROM mavue

(renvoie toutes les colonnes de la table recette)

SELECT * FROM mavue

WHERE titre LIKE 'C%'

(renvoie seulement les recettes dont le titre commence par un C)
